define a function that no return type

no parameters

count from 1 to 100

any time it encounters a 3, print cat,

if it encounteres a multiple of 5, print bat

if it's a multiple of three and five, print catbat

otherwise print out the number itself




catbat() {
    for (int i = 1; i <= 100; i++) {
        if (i % 3 == 0 && i % 5 == 0) {
            System.out.println("catbat");
        } else if (i % 3 == 0) {
            System.out.println("cat")
        } else if (i % 5 == 0) {
            System.out.println("bat")
        } else {
            System.out.println(i);
        }
    }
}
